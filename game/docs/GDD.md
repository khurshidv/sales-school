# Sales School RPG — Game Design Document

## 1. Обзор

**Концепт:** RPG визуальная новелла для тренировки навыков продаж. Игрок выбирает персонажа (м/ж), выбирает сферу продаж, и проживает 5 рабочих дней: к нему приходят клиенты с разными характерами, он выбирает как поступить — каждый выбор ведёт к разным исходам и очкам.

**Целевая аудитория:** Начинающие продавцы в Узбекистане (uz + ru).

**Платформа:** Mobile landscape 16:9 (primary), desktop (secondary).

**Длительность сессии:** 7-10 минут на весь сценарий (5 дней × ~1.5-2 мин/день). 8-12 нод на день. Короткие реплики — 1-2 предложения.

**Core loop:**

```
Выбрать сферу → Начать день → Прочитать ситуацию → Сделать выбор (таймер)
→ Увидеть последствие → Получить очки → Итоги дня → Следующий день
```

Игрок выбирает сферу на старте. Каждая сфера — отдельный сценарий с уникальными клиентами, товарами и диалогами.

## 3. Механики геймплея

### 3.1 Система выборов — 4 стиля

Нет «правильных/неправильных» ответов. Каждый выбор отражает стиль продаж:

| Стиль | Описание | Когда работает | Когда проигрывает |
|-------|----------|---------------|-------------------|
| **Professional** | По инструкции, структурный | Корпоративные клиенты, формальные | Холодный для эмоциональных |
| **Empathetic** | Тёплый, личный, через отношения | Неуверенные клиенты, доверие | Медленный для решительных |
| **Assertive** | Уверенный, прямой, на результат | Спешащие клиенты, закрытие | Агрессивный для пассивных |
| **Adaptive** | Читает ситуацию, подстраивается | Универсальный (если навык есть) | Требует внимания к деталям |

### Связь стиля и архетипа клиента

```
Архетип клиента + Стиль игрока → Модификатор очков

Skeptic + Professional = +20  (уважают компетентность)
Skeptic + Empathetic  = +10  (медленно, но работает)
Skeptic + Assertive   = -5   (сопротивляются давлению)

Impulse Buyer + Assertive   = +20  (совпадает с энергией)
Impulse Buyer + Empathetic  = +5   (замедляет — этично, но меньше очков)
Impulse Buyer + Professional = +10  (нейтрально)

Negotiator + Professional = +15  (уважают подготовку)
Negotiator + Assertive   = +10  (если подкреплено фактами)
Negotiator + Empathetic  = +5   (не про эмоции)

Committee + Adaptive = +25  (нужно балансировать нескольких ЛПР)
Committee + Professional = +15
Committee + Empathetic = +10
Committee + Assertive = -10 (давление на группу не работает)
```

Максимальный балл даёт **Adaptive** — но для этого нужно читать флаги и контекст из предыдущих нод. Награждает внимательных игроков.

### 3.2 Система таймеров

| Тип | Длительность | Когда используется |
|-----|-------------|-------------------|
| Нормальный | 15 секунд | Стандартные выборы |
| Давление | 10 секунд | Возражения, переговоры |
| Критический | 5 секунд | Кульминационные моменты |

**При истечении таймера:** Переход к ноде `fumbled_response` — игрок мямлит, клиент теряет доверие. Штраф -10 очков + потенциальная потеря жизни (на climax-выборах).

### 3.3 Система жизней

| Параметр | Значение |
|----------|----------|
| Стартовые жизни | 3 за сценарий |
| Максимум | 5 |
| Потеря жизни | Рейтинг F за день ИЛИ expiry climax-таймера |
| Получение жизни | Рейтинг S за день |
| Game Over | 0 жизней → рестарт с Day 1 текущего сценария |

**UX жизней:**
- Отображение: сердечки (не цифры) — эмоциональнее
- Потеря: тряска экрана + пульс + красная вспышка
- Получение: золотое свечение + звук торжества
- При 1 жизни: сердце пульсирует как предупреждение

### 3.4 Система комбо

| Подряд правильных | Множитель |
|-------------------|-----------|
| 1-3 | ×1.0 (нет бонуса) |
| 4 | ×1.5 |
| 5+ | ×2.0 |

«Правильный» = выбор, дающий ≥15 очков (из возможных). Серия обрывается при выборе с ≤5 очков.

## 4. Скоринг

### 4.1 Семь измерений

Каждый выбор влияет на 1-3 измерения. Итоговый балл за день — взвешенная сумма:

```typescript
DayScore = (Rapport × 0.20) + (Discovery × 0.15) + (Expertise × 0.15)
         + (Persuasion × 0.20) + (Timing × 0.15) + (Opportunity × 0.05)
         + (Empathy × 0.10)
```

| Измерение | Вес | Как набирать |
|-----------|-----|-------------|
| **Rapport** | 0.20 | Правильное приветствие, активное слушание, запоминание деталей клиента |
| **Discovery** | 0.15 | Открытые вопросы, не прыгать к презентации |
| **Expertise** | 0.15 | Правильные технические ответы, уверенная презентация |
| **Persuasion** | 0.20 | Работа с возражениями, рефрейминг, аргументация |
| **Timing** | 0.15 | Правильный момент для закрытия, правильная техника |
| **Opportunity** | 0.05 | Допродажи, кросс-селл, замеченные доп. потребности |
| **Empathy** | 0.10 | Считывание языка тела, адаптация стиля |

### 4.2 Рейтинги

| Рейтинг | Диапазон | Значение |
|---------|----------|----------|
| **S** (Идеально) | 90-100% от target | Мастерский уровень — нашёл оптимальный путь |
| **A** (Отлично) | 75-89% | Профессиональный уровень |
| **B** (Хорошо) | 60-74% | Компетентно, но есть рост |
| **C** (Средне) | 40-59% | Проходимо, ключевые навыки нужно подтянуть |
| **D** (Слабо) | 20-39% | Трудности — пересмотреть основы |
| **F** (Провал) | <20% | Потерял клиента → потеря жизни |

### 4.3 Модификаторы

| Модификатор | Условие | Эффект |
|-------------|---------|--------|
| Speed Bonus | День пройден быстрее лимита | +10% к баллу |
| Combo ×3 | Достигнут множитель комбо | Применяется к отдельным выборам |
| First Try | День пройден без ретрая | +15% к баллу |
| No Hints | Не использована система подсказок | +5% к баллу |
| Perfect Timing | Все таймерные выборы за первые 50% времени | +10% к баллу |
| Streak | Дни подряд без провала | +5% за день (макс +25%) |

### 4.4 Антигейминг

- Минимальное время на день: 30 секунд (защита от прокликивания)
- Рандомизация порядка вариантов при ретрае (защита от запоминания позиций)
- Вариативность: ±5% к баллу (нельзя получить точно такой же балл на тех же выборах)

## 8. Социальные системы

### 8.1 Лидерборд

| Колонка | Источник | Сортировка |
|---------|---------|------------|
| Ранг | Вычисляемый | По возрастанию |
| Имя | `players.display_name` | — |
| Уровень | `players.level` | — |
| Общий балл | Сумма лучших баллов по сценариям | По убыванию (primary) |
| Сценариев пройдено | Количество уникальных | — |
| Лучший рейтинг | Макс рейтинг за день | — |

### 8.2 Мотивация

- Показывать ранг игрока даже если он #847
- Подсвечивать ближайших соперников: «Обгони Алишера — до него 15 очков!»
- **Weekly лидерборд** (отдельно от all-time) — каждую неделю новый шанс
- Награда для top 10 weekly — эксклюзивное достижение
- При входе: «Ты поднялся на 12 позиций за эту неделю!»

### 8.3 Социальное сравнение (Supabase Realtime)

- Живые обновления когда другой игрок обгоняет тебя
- Уведомление: «Нодира только что получила S на День 3 Автосалона!»
- После дня: «73% игроков выбрали иначе» (анонимно)

## 10. Реиграбельность

### Почему переигрывать?

1. **Разные пути** — минимум 2 жизнеспособных пути на день, разные баллы
2. **Скрытый контент** — варианты выбора видны только при определённых флагах
3. **Стилевой челлендж** — «Пройди только Professional стилем» → достижение
4. **Идеальный проход** — S на каждом дне подряд
5. **Другая сфера** — 4 разных сценария с разными клиентами и товарами

### Что даёт реиграбельность каждому дню

- 1 скрытый вариант диалога (нужен флаг из предыдущего дня)
- 1 «секретная концовка» (необычная комбинация выборов)
- 2+ путей к высокому баллу (разные стили, похожий итог)

**Правила экрана:**
- Всегда показывать разбивку по измерениям (видно ГДЕ улучшаться)
- Самое сильное измерение — зелёным, самое слабое — жёлтым
- Совет по самому слабому измерению: «Попробуй задать открытый вопрос вместо...»
- Тизер следующего дня (хук любопытства)
- Кнопка «Переиграть» всегда доступна (без штрафа)
- При S-ранге: конфетти + особая рамка
