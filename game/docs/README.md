# Game Docs

Здесь хранится вся документация по RPG-игре. Заполняется ДО написания кода.

## Структура

- `GDD.md` — Game Design Document (механики, формулы, достижения, правила)
- `scenarios/` — Полные сценарии со всеми ветвлениями на uz + ru
  - `car-dealership.md` — Автосалон (первый сценарий)
- `asset-map.md` — Карта ассетов (изображения, видео, размеры, стиль)

## Порядок работы

1. Сначала — GDD.md (все механики и формулы)
2. Затем — полный сценарий первого продукта (все дни, все ветвления, все очки)
3. Затем — карта ассетов
4. Ревью: все пути проходимы, баланс очков, нет тупиков
5. Только после этого — код
