## Стиль-гайд для AI генерации

### Персонажи
- Стиль: полуреалистичный, современный. Узбекская внешность.
- Фон: прозрачный (PNG)
- Освещение: студийное, мягкие тени
- Одежда: соответствует описанию персонажа
- Эмоции: чёткие, читаемые на маленьком экране

### Фоны
- Стиль: реалистичный с лёгкой стилизацией
- Палитра: светлая для шоурума, тёплая для VIP, прохладная для парковки
- Детали: логотип Chevrolet видно, машины на фоне узнаваемы
- Формат: JPG, 1920×1080

### Автомобили
- Стиль: фотореалистичный
- Ракурс: 3/4 front (как в каталоге)
- Фон: прозрачный (PNG)
- Цвета: из реального каталога Chevrolet UZ
