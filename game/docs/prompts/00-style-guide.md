# Style Guide — GTA V Loading Screen Art

## Базовый стиль (вставлять в каждый промпт)

```
GTA V loading screen art style, semi-realistic digital illustration,
graphic novel aesthetic, bold cinematic lighting, strong color contrast,
Rockstar Games concept art quality, clean sharp outlines, photorealistic
textures with illustrated feel, dramatic composition, vibrant slightly
desaturated color palette, deep shadows, warm golden tones
```

---

## Инструменты

| Тип | Инструмент | Параметры |
|-----|-----------|-----------|
| Персонажи, фоны, авто | **Gemini Imagen (nanobanana)** | 1920×1080 или 512×768 |
| Катсцены-видео | **Kling 3.0** | 1920×1080, 5-7 сек |
| Открывающие сцены | **Veo3** | 1920×1080, до 8 сек |

---

## Обязательные теги стиля

### Для ПЕРСОНАЖЕЙ (character sprites):
```
GTA V loading screen portrait style, half-body shot, transparent background,
dramatic side lighting, strong shadows on face, semi-realistic illustration,
Rockstar Games art style, clean outlines, 512x768 resolution, PNG format,
Uzbek Central Asian features, cinematic quality
```

### Для ФОНОВ (backgrounds):
```
GTA V environment art style, wide establishing shot, 16:9 landscape,
cinematic composition, warm golden hour lighting, detailed urban environment,
Tashkent Uzbekistan setting, semi-realistic illustration, Rockstar Games
concept art quality, 1920x1080
```

### Для АВТОМОБИЛЕЙ (cars):
```
GTA V vehicle loading screen style, 3/4 front view, dramatic studio lighting,
semi-realistic illustration, clean sharp outlines, reflection on surface,
cinematic quality, transparent background or gradient, PNG format
```

### Для ВИДЕО (Kling / Veo3):
```
GTA V cinematic opening sequence style, slow dramatic camera movement,
warm golden tones, semi-realistic illustration aesthetic, cinematic quality,
no UI elements, 16:9 aspect ratio, 24fps
```

## Важные правила для промптов

1. **Узбекская внешность** — всегда указывать: `Uzbek/Central Asian ethnicity, realistic facial features`
2. **Без текста** — добавлять: `no text, no watermarks, no logos`
3. **Качество** — всегда: `highly detailed, 8K quality, professional concept art`
4. **Стиль** — не забывать: `GTA V loading screen art style` в начале каждого промпта
5. **Негативный промпт** для Imagen: `anime, cartoon, chibi, 3D render, photograph, blurry, low quality, distorted`
