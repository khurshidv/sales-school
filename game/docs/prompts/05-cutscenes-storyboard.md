---

## Как использовать этот файл

Каждая катсцена содержит:
- **Первый кадр** → генерировать в Gemini Imagen (статичное изображение)
- **Последний кадр** → генерировать в Gemini Imagen (статичное изображение)
- **Промпт для Kling 3.0** → использовать оба кадра как start+end frame
- **Промпт для Veo3** → полное описание движения (если Kling недостаточно)

## Порядок генерации (рекомендуемый)

1. Сначала генерировать start и end кадры в **Gemini Imagen**
2. Загрузить оба кадра в **Kling 3.0** (Image-to-Video)
3. Если нужно больше контроля — использовать **Veo3** с промптом

### Параметры Kling 3.0:
- **Mode:** Professional
- **Duration:** 5-8 секунд
- **Aspect:** 16:9
- **Camera:** Cinematic smooth dolly / push
- **Style:** Cinematic

### Параметры Veo3:
- **Quality:** High
- **Style:** Cinematic
- **Frame rate:** 24fps
- **Aspect:** 16:9
