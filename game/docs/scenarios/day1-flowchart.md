## Максимальный маршрут к скрытой концовке

1. **Выбор 1-А** — к обоим → `addressed_both` + rapport+15, empathy+5
2. **Выбор 2-А** — баланс → `balanced_both` + empathy+15, persuasion+10
3. Тест-драйв → rapport+5
4. **Выбор 4-В** — молчать → rapport+10
5. Нилуфар упоминает годовщину → `knows_anniversary`
6. **Выбор 5-Б** — подарок на годовщину → `anniversary_surprise` + opportunity+20, empathy+10

**Итого: rapport+30, empathy+30, persuasion+10, opportunity+20, timing+0 = ~90 очков**
